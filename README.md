# Validador y Generador de RUT Chileno 🇨🇱

Este es un proyecto simple en Python diseñado para manejar la lógica del Rol Único Tributario (RUT) en Chile. Es ideal para ser utilizado como base en sistemas que requieran validación de usuarios o generación de datos de prueba.

## ✨ Características

- **Validación:** Comprueba si un RUT es válido mediante el algoritmo Módulo 11.
- **Generación:** Crea RUTs aleatorios válidos matemáticamente.
- **Modularidad:** Lógica separada de la interfaz de usuario.

## 🚀 Cómo ejecutar

1. Clona este repositorio.
2. Ejecuta el archivo principal:
   ```bash
   python main.py
   ```
